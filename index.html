Twitch consumption data visualization

Pour consulter notre avancement : wiki

Nous allons récupérer les données (nombre de viewer sur chaque stream, etc.) sur les streams sur une durée d'une semaine, tous les quarts d'heures. Ainsi nous aurons assez d'informations pour pouvoir comparer la consommation selon les heures et les jours de la semaine.

Nous avons récolté les noms de chaînes des 100 streamers français les plus populaires au sein de ce fichier.

Nous avons ensuite créé un script python afin de récupérer l'identifiant de chaque chaîne sur le site Twitch, ce qui nous sera utile lors de nos requêtes. Les identifiants sont automatiquement corrélés au streamer correspondant au sein de ce fichier csv.
